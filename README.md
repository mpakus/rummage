
Rummage
=====

[![Circle CI](https://circleci.com/gh/mpakus/rummage/tree/master.svg?style=svg)](https://circleci.com/gh/mpakus/rummage/tree/master)

* search engine

Install & run
----
> bundle install

> rails s

type `http://localhost:3000/` in your browser and start to search!

Tests
---
> rspec

Rubocop
--- 
> rubocop -a
